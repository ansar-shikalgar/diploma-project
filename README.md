# E-Commerce Website

An E-Commerce Website developed using PHP, MySQL, HTML, CSS, Bootstrap, and JavaScript.
This project allows users to browse products, manage carts, create accounts, and place orders through a simple and user-friendly interface.

# Project Objective

The objective of this project is to design and develop a dynamic online shopping platform that demonstrates:

Server-side scripting using PHP

Database connectivity with MySQL

User authentication

Cart management system

This project is developed as part of an academic / learning project.

# Installation & Setup
      *Follow these steps to run the project locally:

1️⃣ Install XAMPP
      Download and install XAMPP from the official website.
      
2️⃣ Start Services
      Start:
         *Apache
         *MySQL
         
3️⃣ Import Database
      *Open phpMyAdmin
      *Create a database named:
         ecommerce
      *Import the file:
         database/ecommerce.sql
         
4️⃣ Move Project Files
      *Copy the project folder to:
         xampp/htdocs/

5️⃣ Run the Project
      *Open your browser and go to:
         http://localhost/e-commerce_website/
